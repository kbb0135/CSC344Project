Chapter Two

Introduction to HTML Classes and ID.

  • The HTML class attribute is used to specify a class for an HTML element.
  • The class name is case sensitive.
  • The class attribute is often used to point to a class name in a style sheet.
  • The class attribute is also used by a Javascript to access and manipulate elements with specific class name.

    Uses of HTML ID tags:

  • The id attribute is used to specify the unique ID for an element of the HTML document.
  • It also allocates the unique identifier which is used by the CSS and the JavaScript for performing certain tasks.
  • The identifier must be unique across the page.
  • The id is a global attribute that can be applied to any HTML element.
  • In HTML you can only have one ID per element but you can indeed have more than one class.
  • In CSS, the id attribute is used using the # symbol followed by id.
