Introduction to HTML Forms and Attributes

  • An HTML form is used to collect user input.
  • The "form" element is a container for different types of input elements such as: text fields, checkboxes, radio buttons, submit buttons, etc.
  • The "action" attribute in form defines the action to be performed when the form is submitted.

    Uses of basic HTML forms and attributes:

  • The "action" scripts prepares or makes ready to process the passed data.
  • There are different types of form controls that can be used to collect data such as: Text Input Controls, Select Box Controls, Clickable Buttons, and Submit/Reset Buttons.
  • The "type" attribute indicates the type of input control.
  • The "name" attribute is used to give a name to the contol which is sent to the server to be recognized.
  • The "size" allows to specify the width of the text-input control in terms of characters.
  • The "maxlength" attribute allows to specify the maximum numbers of characters a user can enter in a text box.
