Chapter One

Intorduction to HTML

  • HTML stands for Hyper Text Markup Language.
  • HTML is used for creating web pages through different HTML elements.
  • HTML has many tags used to define different elements.

    • Uses of basic HTML tags:

  • head: The head cretes the title and allows to use CSS file.
  • body: Helps to create all the content in the website.
  • h1: Creates heading for the web page.
  • p: Creates paragraph for the webpage.
  • li: Defines a list item.
  • ol: Creates ordered list.
  • ul: Creates unordered list.
  • a: has many tags used to define different elements.
