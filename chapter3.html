    Uses of basic HTML images and styles:

  • The HTML "img" tag is used to embed an image in a web page.
  • HTML uses the width and height attributes or the CSS width and height properties to define the size of an image.
  • To use an image as a link, put the "img" tag inside the "a"tag.
  • HTML also uses the CSS property to let the image float to the right or to the left of a text.
  • The required "alt" attribute provides an alternate text or caption for an image.
  • The width and height attributes always define the width and height of the image in pixels.
  • The "src" specifies the path to the image.
